Styling Story for Makers

화면을 읽기 좋게 만들려는 실험과 도구의 발자국

종이 인쇄물로 시작한 타이포그래피, CSS가 등장하며 생긴 표현의 자유, 그리고 반응형 레이아웃과 컨테이너 쿼리까지. “어떻게 하면 화면을 덜 답답하게 만들까?”라는 질문에 답한 사건을 모았습니다.

연도 버튼을 누르면 사건에 참여한 디자이너와 개발자의 이야기가 펼쳐집니다. 복잡한 개념은 모달 본문에서 풀어 설명합니다.

연대기를 누르면 같은 화면에서 상세 이야기를 읽을 수 있습니다.

1980s

데스크톱 퍼블리싱의 시작

타이포그래피가 디지털로 옮겨 가면서, 화면에서도 인쇄물 수준의 조형을 하려는 시도가 이어졌습니다.

1990s

CSS 탄생과 레이아웃의 첫걸음

웹 서식 언어를 만들려는 제안이 나왔고, CSS1 사양이 채택되며 브라우저가 스타일 규칙을 공유하기 시작했습니다.

2000s

표현 기법과 도구가 폭발하다

디자이너는 실험을 통해 CSS 가능성을 증명했고, 전처리기가 등장하며 스타일 재사용이 쉬워졌습니다.

2010s

반응형과 현대 레이아웃

모든 화면에서 읽기 좋은 구성을 목표로, 반응형 접근과 Flexbox, Grid가 실무에 자리 잡았습니다.

2020s

컨테이너 기반 레이아웃과 최신 CSS

브라우저가 컨테이너 쿼리, 캐스케이드 레이어, 네이티브 네스팅을 지원하며 스타일 시스템이 성숙해졌습니다.
